# AI 营销文案生成器

> 输入产品信息，一键生成小红书爆款文案 + 朋友圈种草文案

[在线演示](https://zqleslie.github.io/ai-copywriter-cn/)

## 使用方法

1. 输入产品名称
2. 填写卖点关键词（用空格或逗号分隔）
3. 选择文案风格（热情种草/专业评测/情感故事/幽默搞笑/高级轻奢）
4. 点击「一键生成文案」
5. 点击「📋 复制」按钮直接复制文案

## 特色功能

- 🎨 5 种文案风格自由切换
- 📱 手机端完美适配
- 📋 一键复制，直接发布
- 🔒 纯前端运行，数据不上传
- ⚡ 零等待，即时生成

## 技术栈

- 纯 HTML + Tailwind CSS + Vanilla JS
- 零依赖，零构建
- 内置文案模板引擎
